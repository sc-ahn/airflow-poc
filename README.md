# Airflow PoC

```bash
> make help

Usage:
  make <target>

Targets:
  help                  지금 보고계신 도움말
  prepare               초기화단계에 필요한 작업
  down                  컨테이너 종료
  up                    컨테이너 실행
  init-airflow-db       Airflow DB 초기화
  info                  Airflow 정보
  get-pools             Airflow Pools 정보
make build && make up 실행후
localhost:8080에 접속하세요
ID: airflow
PW: airflow
```
